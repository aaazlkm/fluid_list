# fluid_list_example

A demo for [`fluid_list`](../): spring-animated show / hide / move and drag-to-reorder.

- A **vertical task list** — add, shuffle, and delete cards to see the spring enter, move, and exit
  animations; long-press a card to reorder it.
- A **drag-mode toggle** in the app bar — switch between long-pressing the whole card and dragging a
  dedicated `FluidListDragHandle` grip.
- A **horizontal tag strip** built with `axis: Axis.horizontal`, reorderable and removable.

## Run

```sh
fvm flutter run
```
